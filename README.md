# Quant Backtesting System

This project implements a simple Moving Average Crossover trading strategy using historical stock data.

## Strategy Logic

- 20-day moving average (short-term)
- 50-day moving average (long-term)
- Buy when MA20 > MA50
- Sell when MA20 < MA50

## Performance Metrics

- Cumulative Return
- Maximum Drawdown
- Sharpe Ratio
- Equity Curve Visualization
